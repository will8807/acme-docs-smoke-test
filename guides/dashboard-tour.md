# Dashboard Tour

This guide introduces the Acme dashboard and its main sections.

## Overview

The Acme dashboard gives you a central place to manage your projects, monitor
usage, and access the developer tools.

![Acme Dashboard](../assets/hero.png)

The dashboard is divided into three primary areas:

- **Projects panel** — lists all your active projects with live status
- **API usage** — shows request volume and error rate for the last 30 days
- **Quick actions** — shortcuts to create a project, rotate keys, or invite a team member

## Navigation

Use the sidebar to move between sections:

| Section       | Description                          |
|---------------|--------------------------------------|
| Projects      | Manage and deploy your projects      |
| API Keys      | Create and revoke access credentials |
| Logs          | Inspect recent request logs          |
| Settings      | Account and billing configuration    |

## Status Indicators

Each project card shows a colour-coded status badge:

- **Green** — running normally
- **Yellow** — degraded or pending deployment
- **Red** — stopped or error state
