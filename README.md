# Acme Docs

Welcome to the **Acme** documentation hub. This repository contains the full product
documentation for the Acme platform, including getting-started guides, API references,
and architecture overviews.

## Quick Start

```bash
npm install @acme/sdk
acme login
acme init my-project
```

## Structure

| Path | Contents |
|------|----------|
| `docs/` | Core reference docs |
| `guides/` | Step-by-step tutorials |
| `assets/` | Images and diagrams |
